# address-book-app

    Address book SPA on ES6, build with Webpack2.

## Installation

    $ npm install

    #or

    $ yarn install

## Run in development

    $ npm start

## Build for production

    $ npm run build

## Preview of the production build

    $ npm run prod

## Tests

    $ npm run test
